# Customer Segmentation using K-Means Clustering

## 📌 Problem Statement

Retail chains struggle to personalize offers because they lack insights into customer behavior. This project uses clustering to group customers and provide targeted recommendations.

---

## 🧠 Approach

We applied K-Means Clustering to segment customers based on:

* Annual Income
* Spending Score

---

## ⚙️ Tech Stack

* Python
* Pandas
* Matplotlib
* Scikit-learn

---

## 📊 Steps Performed

1. Data loading using Pandas
2. Data cleaning and inspection
3. Feature selection
4. Applied K-Means clustering
5. Visualized clusters
6. Used Elbow Method to find optimal clusters
7. Interpreted customer segments

---

## 📈 Results

Customers were grouped into 3 clusters:

* High-value customers → Premium offers
* Low spenders → Discounts
* Medium customers → Personalized recommendations

---

## 💡 Business Impact

* Improves personalization
* Enhances customer engagement
* Increases sales

---
